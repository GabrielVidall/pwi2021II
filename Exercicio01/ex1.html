    Linguagens utilizadas

  • HTML

    • HTML é uma linguagem de marcação de hipertexo, toda o organização de um pagina web como o cabeçalho, título, parágrafos, imagens e muitos outros é feito pelo html.

  • CSS

    • CSS é uma linguagem de estilos (cuida da parte estética do texto), é ele que da cor ao texto na web, muda a fonte e diversas outras coisas.
